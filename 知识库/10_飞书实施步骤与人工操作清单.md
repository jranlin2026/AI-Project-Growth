## 你需要手动完成

1. 在飞书项目群添加自定义机器人，复制 Webhook 和签名密钥。
2. 在飞书任务里创建任务清单：`AI商业IP增长项目每日作战`。
3. 在飞书多维表格创建两个表：每日作战日报、视频数据记录。
4. 在飞书开放平台创建企业自建应用，获取 `app_id` 和 `app_secret`。
5. 给应用开通多维表格读取/写入权限；后续如果自动创建任务，再开通任务权限。
6. 在项目根目录创建 `.env`，填入真实密钥和表格 ID。

## 验收顺序

第一步：测试飞书群提醒。

```powershell
node scripts/feishu/send-webhook.mjs --text "AI商业IP增长项目飞书机器人测试"
```

第二步：生成今日任务。

```powershell
node scripts/feishu/generate-daily-task.mjs
```

第三步：生成并推送今日任务。

```powershell
node scripts/feishu/generate-daily-task.mjs | node scripts/feishu/send-webhook.mjs
```

第四步：读取日报表。

```powershell
node scripts/feishu/read-bitable-records.mjs
```

## 管理架构确认

最终运行方式：

```text
增长负责人（Codex）
  -> 分配任务给内容管理、文案、剪辑、发布、数据分析智能体
  -> 智能体产出脚本、剪辑说明、发布策略、复盘口径
  -> 需要真人执行的事项进入飞书任务
  -> 组员执行后在飞书任务更新状态，并在多维表格填数据
  -> Codex 读取数据，复盘当天，生成明天任务
```
