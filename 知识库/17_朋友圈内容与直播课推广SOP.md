# 朋友圈内容与直播课推广 SOP

## 一句话结论

AI 商业 IP 增长项目不能只做短视频内容。短视频负责公域获客，朋友圈负责私域信任、活动预热和转化成交。直播课推广必须有独立朋友圈文案节奏，而不是临时发一条硬广。

## 内容分工

```text
短视频 = 让陌生老板看见你
朋友圈 = 让加到微信的人相信你
直播课 = 给潜在客户一个集中理解和行动的理由
```

朋友圈不是“复制短视频文案”，而是把短视频筛进来的线索进一步教育、建立信任、制造报名动作。

## 朋友圈内容类型

| 类型 | 目的 | 适用场景 |
| --- | --- | --- |
| 观点型 | 建立认知和专业判断 | 日常发 |
| 案例型 | 建立真实感和结果感 | 有客户反馈、项目过程时 |
| 过程型 | 建立持续做事的信任 | 备课、交付、复盘时 |
| 资料型 | 承接短视频 CTA | 发《老板AI落地实战地图》时 |
| 直播课预热型 | 让用户知道为什么要来 | 直播前 3-5 天 |
| 直播课临门型 | 提醒报名和到场 | 直播当天 |
| 直播课复盘型 | 二次转化未到场用户 | 直播后 1-2 天 |

## 直播课朋友圈节奏

一场直播课至少准备 7 条朋友圈：

| 时间 | 内容 | 目的 |
| --- | --- | --- |
| T-3 天 | 痛点预热 | 让老板意识到问题 |
| T-2 天 | 方法预告 | 告诉用户这场课能解决什么 |
| T-1 天上午 | 案例/过程 | 建立真实感 |
| T-1 天晚上 | 报名提醒 | 拉第一波报名 |
| 当天上午 | 主题提醒 | 明确时间和收益 |
| 开播前 2 小时 | 临门提醒 | 促进行动 |
| 直播后 | 复盘与资料承接 | 承接未到场和二次转化 |

## 朋友圈文案必须包含

每条直播课朋友圈，Codex 必须先写清楚：

1. 目标人群：这条写给哪类老板。
2. 核心痛点：他们为什么会停下来。
3. 这条朋友圈的目的：预热、报名、提醒、复盘还是成交。
4. 正文文案：口语化，不写成海报广告。
5. 配图建议：截图、备课图、白板图、课程大纲、交付资料。
6. CTA：评论、私信、扫码、进群、预约。
7. 风险提醒：不能夸大效果，不能承诺收益。

## 标准文案结构

### 结构 A：痛点预热

```text
很多老板现在做 AI，最大的问题不是不会用工具。

而是公司里面到底哪件事最值得先用 AI 改，没想清楚。

你可能收藏了很多工具，员工也试了几个软件，但获客、销售、客服、内容生产这些真正贵的问题，并没有变轻。

我这次直播课会讲一个很简单的判断方法：
先找最贵的问题，再拆流程，最后才选工具。

想听的可以私信我，我发你时间。
```

### 结构 B：方法预告

```text
这次直播课我不会讲一堆 AI 工具清单。

我会讲老板真正能用上的 3 件事：

1. 怎么判断公司最适合先落地 AI 的场景
2. 怎么把获客、销售、客服、内容这些流程拆出来
3. 怎么让员工真的用起来，而不是老板一个人兴奋

如果你是企业老板，想知道 AI 到底怎么和业务结合，这场会比较适合。

想要直播时间，私信我。
```

### 结构 C：临门提醒

```text
今晚这场 AI 落地直播课，我会尽量讲得接地气一点。

不讲玄的，不讲工具合集，重点讲老板怎么判断：

- 公司现在最贵的问题是什么
- 哪个流程最适合先用 AI 改
- 怎么避免员工不用、老板白忙

已经预约的，记得提前进来。
还没拿到入口的，私信我。
```

### 结构 D：直播后复盘

```text
昨晚直播里，大家问得最多的不是“哪个工具最好用”。

反而是：

- 我这个行业适不适合用 AI？
- 销售跟进能不能先改？
- 员工不会用怎么办？
- 短视频获客能不能用 AI 批量做？

这说明老板真正关心的不是工具，而是业务结果。

我把昨晚讲到的《老板AI落地实战地图》整理好了，需要的私信我。
```

## 飞书任务写法

朋友圈任务不能只写“发朋友圈”。

正确任务必须写清楚：

```text
任务：发布直播课朋友圈预热文案 1 条
负责人：N哥/阿浩/小彭
截止时间：今天 17:30

发布内容：
【完整朋友圈文案】

配图建议：
1. 直播课主题图
2. 备课截图
3. 《老板AI落地实战地图》局部截图

发布后反馈：
1. 是否已发
2. 朋友圈截图
3. 有几个人私信/评论
4. 真实问题是什么
```

## 数据记录

朋友圈内容至少记录：

| 字段 | 说明 |
| --- | --- |
| 日期 | 发布日期 |
| 类型 | 观点/案例/资料/直播课预热/直播课提醒/复盘 |
| 文案主题 | 这条朋友圈讲什么 |
| CTA | 私信/评论/扫码/进群 |
| 发布人 | 谁发 |
| 截图 | 朋友圈截图 |
| 私信数 | 多少人主动问 |
| 评论数 | 多少人评论 |
| 真实问题 | 用户问了什么 |
| 是否带来报名 | 是 / 否 / 不确定 |

## Codex 每天要做什么

如果当天有直播课、社群活动、资料领取或转化动作，Codex 必须在每日执行包里额外生成：

1. 今日朋友圈内容目标。
2. 朋友圈完整文案。
3. 配图建议。
4. 发布时间。
5. 谁发布。
6. 发布后怎么反馈。
7. 私域数据看什么。

## 当前决策

后续每日执行包不只包含短视频脚本，也要包含私域朋友圈内容。尤其是直播课推广，必须提前 3-5 天形成朋友圈预热节奏。
