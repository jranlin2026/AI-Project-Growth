# 飞书日报与视频数据字段

## 每日作战日报表

建议表名：`AI商业IP增长项目_每日作战日报`

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| 日期 | 日期 | 日报日期 |
| 填写人 | 人员 / 文本 | 林总、编导A、编导B |
| 角色 | 单选 | 林总 / 编导A / 编导B |
| 今日任务完成情况 | 多行文本 | 今天完成了什么 |
| 未完成事项 | 多行文本 | 没完成什么 |
| 未完成原因 | 多行文本 | 原因 |
| 明日建议 | 多行文本 | 对明天任务的建议 |
| 需要支持 | 多行文本 | 需要谁支持 |
| 任务状态 | 单选 | 已完成 / 部分完成 / 未完成 |

## 视频数据记录表

建议表名：`AI商业IP增长项目_视频数据记录`

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| 日期 | 日期 | 发布日期 |
| 平台 | 单选 | 抖音 / 视频号 / 小红书 |
| 账号 | 单选 | 账号A / 账号B / 账号C |
| 脚本ID | 文本 | 例如 S001 |
| 视频标题 | 文本 | 实际发布标题 |
| 内容类型 | 单选 | 认知纠偏 / 案例拆解 / 实操演示 / 降本增效 / 销售转化 |
| 开头钩子 | 多行文本 | 前 3 秒钩子 |
| 视频链接 | URL | 发布链接 |
| 播放量 | 数字 | 播放 |
| 点赞数 | 数字 | 点赞 |
| 评论数 | 数字 | 评论 |
| 实战地图评论数 | 数字 | 评论关键词数量 |
| 私信数 | 数字 | 私信数量 |
| 进群人数 | 数字 | 进群数量 |
| 老板型线索数 | 数字 | 明确老板/负责人线索 |
| 真实业务问题 | 多行文本 | 评论/私信/群里的真实问题 |
| 初步判断 | 单选 | 加码 / 观察 / 调整 / 淘汰 |

## 飞书任务结构

每天一个父任务：

```text
YYYY-MM-DD AI商业IP增长项目 Day X 作战任务
```

子任务：

| 子任务 | 负责人 | 截止时间 |
| --- | --- | --- |
| 林总：完成今日出镜拍摄 | 林总 | 17:00 |
| 编导A：完成今日脚本和明日选题 | 编导A | 17:00 |
| 编导B：完成剪辑、发布、评论承接和数据记录 | 编导B | 17:30 |
| 编导B：填写视频数据记录初填 | 编导B | 17:30 |

补充规则：

- 人工工作时间为 09:00-18:00，任务默认在 17:30 前完成状态更新和数据初填。
- 隔夜数据次日 09:30 补充。
- 彭涵奕每周六休息，周六不分配编导 B 任务。
- 朱勇浩每周日休息，周日不分配编导 A 任务。
